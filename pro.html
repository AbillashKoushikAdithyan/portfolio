SMART CRADLE SYSTEM

The infant in the cradle is monitored in real time and alert is given to the parentâ€™s mobile phone as a notification whenever necessary. The Cradle can be monitored and control using the mobile application. Cradle has the feature of live monitoring, cry detection using sound sensor and to find infants rigorous movement using PIR motion sensor. The ideal temperature for the infant to sleep is been monitored using DHT11 Temperature and Humidity sensor. The cradle swinging mechanism is a unique feature through which the cradle can be swing with a press of the button in the app. The mobile application is been developed using Android Studio. Raspberry Pi and Arduino Mega are the Micro-controllers used in the project

Sensitive documents are kept without the harm of being tampered.

E-IDENTITY USING COMBINED CRYPTOGRAPHY

This project provides the flexibility of passing information by implementing the Combined Cryptography encryption techniques through a user friendly Graphical user interface in such a way that reduces the manual overhead of verification and paper work of identity documents possessed by an individual. The system makes use of Combined Cryptography (Advanced Encryption Standard and Triple Data Encryption Standard) which makes it impossible for the hackers to break into the system. Authentication and non-repudiation is maintained with the help of Digital Signature. Therefore sensitive documents are kept without the harm of being tampered.

HOME AUTOMATION USING NODE MCU AND GOOGLE ASSISTANT

The aim of the project is to control home appliances using Google assistant. Using IFTTT two applets are created to switch ON and OFF lights in two rooms. The command from IFTTT is send to Adafruit IO. The lights are connected to Node MCU through relay module. When the command is said through Google assistant, the corresponding lights would turn ON or OFF depending on the user command.
